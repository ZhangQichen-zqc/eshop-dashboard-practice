# 初步业务发现报告

> 基于 2024-04 至 2026-03 共 24 个月的经营数据

## 一、经营规模概览

课程商城在两年间累计实现 GMV 1.56 亿元，订单 10.8 万笔，服务 1.8 万名买家。客单价 1,444 元，属于中高端消费客群。整体毛利率 50.8%，盈利能力健康。退款率 7.75% 处于电商行业合理范围（通常 5-15%）。

## 二、GMV 月度趋势

月度 GMV 呈现清晰的三段式走势：
- **2024 Q2-Q3（起步期）**：月均 GMV 约 200-400 万，波动较大，符合新商城上线初期的特征。
- **2024 Q4-2025 Q2（爆发期）**：GMV 快速攀升至月均 800-1,000 万，2025 年 6 月达到峰值约 1,100 万。这一阶段叠加了营销活动密集投放和用户规模增长的双重驱动。
- **2025 Q3-2026 Q1（稳定期）**：GMV 回落并稳定在月均 500-700 万，同时客单价保持平稳（1,400-1,500 元），说明回落主要来自订单量而非单价，可能归因于营销投放力度调整或用户增长放缓。

## 三、渠道结构

六大渠道贡献分化明显：
- **organic（自然流量）**贡献最高，占 26.8%，说明品牌有自然获客能力。
- **search 和 social** 分列第二、三位（23.9% 和 20.2%），合计贡献 44%，是投放的核心阵地。
- **email 和 affiliate** 各占 13.9% 和 10.3%，是稳定的辅助渠道。
- **live（直播）**仅占 4.9%，体量虽小但转化率数据需进一步分析（文档提到直播转化率为平均的 1.35 倍）。

## 四、品类表现

12 个品类中呈现明显的头部集中效应：
- **TOP 3 品类（数码电子、小家电、美妆个护）**合计贡献 GMV 的 51%，是绝对的核心类目。
- 数码电子 GMV 达到 3,580 万，远超第二名小家电（2,274 万），但需关注其毛利率是否偏低（电子类通常毛利较薄）。
- **尾部品类（食品饮料、母婴用品、图书文创、宠物生活）**各占 2.3%-4.7%，合计仅约 17%，可评估是否需调整品类策略。

## 五、初步建议方向

1. **深挖 organic 渠道**：26.8% 的自然流量贡献说明 SEO/品牌建设有效，应进一步分析这批用户的留存和价值。
2. **关注中期 GMV 回落原因**：2025 下半年 GMV 从峰值回落，需排查是用户流失、渠道效率下降还是营销预算缩减。
3. **品类结构优化**：TOP 3 品类集中度过高，需评估尾部品类是需求不足还是供给不足。
4. **转化漏斗诊断**：流量到支付整体转化率 49.39%，但各环节（浏览→加购→结算→支付）的瓶颈需要定位。
5. **营销 ROI 评估**：六大渠道的 GMV 贡献已知，但需结合广告花费 (fact_ads_spend) 计算各渠道 ROAS，找出低效投放。
